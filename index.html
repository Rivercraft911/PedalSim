Notes

Knobs map to real component values. The clipping curve is solved per-sample from the Shockley diode equation (Newton-Raphson) and baked into a WaveShaper for zero-cost real-time processing. The tone stage is a real RC lowpass. Values shown on the Build Sheet snap to standard E12 resistor and capacitor values — what you'd actually buy.
